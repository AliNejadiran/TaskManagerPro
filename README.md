# ✨ Task Manager Pro

A professional task management application built with **Python CustomTkinter**.  
Organize your tasks with priority, category, deadline, countdown timer, and advanced search/filter.

## 🚀 Features

- ✅ Add / Edit / Remove / Complete tasks
- 🎯 Priority levels: High, Medium, Low
- 📂 Categories: Personal, Work, Shopping, Health, Education
- 📅 Deadline with time (YYYY-MM-DD HH:MM)
- ⏳ Countdown timer showing remaining time
- 🔍 Search & Filter by text, priority, category
- 📊 Statistics (Total, Completed, Pending, Today's tasks)
- 🎨 Modern Orange-Gold dark theme
- 💾 Auto-save to JSON file
